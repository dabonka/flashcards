<h1>This is a web application for learning foreign language words</h1>

<h2>Setup</h2>
<pre>
<code>
git clone github.com/dabonka/flashcards.git
bundle install
cp config/database.ci.yml config/database.yml
rake db:migrate
</code>
</pre>

<ul>
<li>Ruby on Rails 4.2</li>
<li>Ruby 2.0</li>
</ul>
